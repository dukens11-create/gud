## üé® Design Guidelines

### Screenshot Design Tips

1. **Show Key Features First**
   - Most important screenshots at the beginning
   - Users may only see first 2-3

2. **Use Annotations**
   - Add text overlays highlighting features
   - Use arrows and callouts
   - Keep text readable (large font)

3. **Real Content**
   - Use realistic data
   - Show actual functionality
   - Avoid lorem ipsum

4. **Consistent Style**
   - Same device frames
   - Matching color scheme
   - Professional appearance

5. **Include Context**
   - Show how features work together
   - Tell a story with screenshots
   - Demonstrate value proposition

### Tools for Screenshots

**Recommended:**
- [Screely](https://screely.com/) - Device mockups
- [App Screenshot Maker](https://appscreenshotmaker.com/)
